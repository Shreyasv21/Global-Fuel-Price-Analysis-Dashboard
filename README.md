# Global-Fuel-Price-Analysis-Dashboard 

## Project Overview

This project is an interactive Excel Dashboard created to analyze global fuel prices across different countries and regions from 2020 to 2026.

The dashboard focuses on understanding how petrol, diesel, and crude oil prices are influenced by taxation policies, regional differences, and crude oil market trends.

This was my first complete Excel dashboard project and helped me strengthen my skills in data analysis, dashboard design, and business interpretation.

_________________
## Key Features

* KPI Cards for Petrol, Diesel, Crude Oil, and Tax Percentage
* Fuel Price Movement vs Crude Oil Trend Analysis
* Petrol vs Diesel Price Comparison by Country
* Average Fuel Tax Rate by Country (Map Visualization)
* Fuel Price Spread Across Regions
* Interactive Slicers for Country, Region, and Timeline

________________

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Map Charts
* Dashboard Design Principles

________________

## Key Insights

* Fuel prices closely follow crude oil market trends
* Taxation significantly impacts final consumer fuel prices
* Regional differences reflect policy and supply chain costs
* Diesel remains relatively cheaper to support logistics
* Fuel spread highlights government pricing strategies
